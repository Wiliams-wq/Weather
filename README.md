
# weather

Weather es una web-app cuya función básica es poder pronosticar el clima en actual de distintas ciudades del mundo. También recibes un pronśtico por los siete días siguientes.

## Tecnologías Usadas  
#### Javascript💛  
#### Vue.js 3 (Compostion API)💚  
#### Vue-Router💚  
#### Vuex💚  
#### SASS💜  
#### Ed-Grid💙
